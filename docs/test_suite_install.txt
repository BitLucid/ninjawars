
Installing the test Suite (optional)
-------------------------
// Install pear if it's not already installed
sudo apt-get install php-pear
sudo pear channel-update pear.php.net
sudo pear channel-update pear.phpunit.de
sudo pear upgrade PEAR
// Install php unit.
sudo pear config-set auto_discover 1
sudo pear install pear.phpunit.de/PHPUnit
// Install selenium bridge
sudo pear install phpunit/PHPUnit_Selenium
// Test to see whether it's installed and working...
phpunit --version

// Optionally, install xdebug
sudo apt-get install php5-xdebug

Download latest version of selenium from here:
http://seleniumhq.org/download/


From the root of your project:
mkdir tests
cd tests

Drop this into phpunit.xml:
<phpunit bootstrap="./TestHelper.php" colors="true">
    <testsuite name="MyProject">
        <directory>./</directory>
    </testsuite>
</phpunit>

Replace MyProject, of course.

Create the TestHelper.php file with the default includes for phpunit testing.
